c언어의 malloc이란 함수를 
메모리를 동적 할당을 하면
변수르 이용해서
피룡한 크기를 계산해
버퍼를 만들 수 있어

실행중 메모리 크기 변경이 필요할 때
realloc(reallocation) 함수를 통해 변경할 수 있다


char *buffer _
= (char*)malloc(변수 + 1{1은 NULL을 위한 것})

fseek() ftell() 같은 함수들과 변수를 이용해
그때 의 크기에 맞춰 버퍼의 크기를 만들 수 있다.

만약 더 많은 데이터를 저장할 버퍼가 필요하면
'realloc'으로 필요한 메모리 사이즈를 변경하면 된다.

// 기존의 malloc()으로 저장했던 데이터는
    realloc()함수로 버퍼 사이즈르 변경해도 자동으로 보존된다.

-----------------------------------------------------------------

대부분의 프로그래밍 언어는 실행중 변경이 가능한 코드들은
'힙(heap)'이란 메모리 공간에 저장이 되어
힙에 저장되는 코드들을 모두 동적 요소가 볼 수 있다.


3. 네트워크에서 정적, 동적이란

인터넷이란 라우터들의 집합체
네트워크 엔지니어
-> 라우터 설정하고 네트워크망을 관리하는 일

만약 와이파이 사용할 대 ip주소가 필요
-> 공유기에서 만든 가상의 ip를 할당받으며
    가상의 ip는 공유기에서 공인 ip로 바귀어 
    언테넛에서 사용이 된다.

// 와이파이에 비번만 인력하명 자동으로 와이파이에
   연결이 되는 이유는
공유기에서 DHCP 라는 기술을 이용해
실시간으로 남은 ip를 접속한 사람들에게
할당했기에 가능했던 것이였습니다


-----------------------------------------------------------------


dynamic(동적, 실시간 상황에 따라 변하는)
DHCP
-> 실시간으로 ip사용중이던 사람이 빠지면 IP를 회수하고,
새로운 사람이 들어오면 여분의 ip를 할당하는 기술(프로토콜)

동적할당(DHCP)을 사용하지 않을 경우
네트워크관리자가 
일일이 남은 ip를 할당했다가,
다수 회수했다가 해야할 것이다.

보통 이런 경우 보안을 위해서 ip를 정적(static)할당을
사용하는 것이며
보통 네트워크 관리자에게 부탁해야 합니다

우리 삶에서 사용되는 와이파이는 대부분 동적기술이라느 것이다
